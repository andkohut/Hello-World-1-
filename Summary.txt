Андрій Когут
GitHub – цікава програма для людей, які бажають отримати можливості створення власних проектів. Спочатку для мене він був важким і незрозумілим, що уповільнювало вивчення його можливостей та інструментів. Згодом, після певного періоду використання, ти звикаєш і починаєш розуміти. Для тих, кому це в новинку, зокрема як я, це був захоплюючий досвід.
Я навчився використовувати github для швидкої співпраці з другом, створення власного проекту. Зокрема, це чудове місце для знайомств з новими людьми та обговорювання тем, якими ви цікавитесь. Беззаперечною перевагою використання github є можливість спільного редагування проектів(текстів), що сутєво економить час, оскільки не має необхідності використання поштових скриньок(пересилання файлів).
